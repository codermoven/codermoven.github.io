hexo github 搭建个人博客

hexo搭建个人博客教程

前提：

拥有github账号,安装好npm

本地构建

操作步骤如下：

  1. 安装hexo-cli: pm install -g hexo-cli
  2. 初始化博客框架：hexo init blog
  3. 进入blog文件夹，安装依赖包：
    • cd blog
    • npm install
  4. 启动hexo服务：hexo server
  5. 浏览器输入：127.0.0.1:4000即可访问

如需写文章，只需要在\blog\source_posts\文件夹下面防止写好的md文件，之后执行hexo g即可在个人站点访问刚才发布的md文件；

如有不明确之处可参见官网：hexo官网

主题安装

hexo提供了众多主题，可以根据个人喜好进行安装，安装过程如下：

  1. 登录hexo官网(https://hexo.io/)，选择主题页面
  2. 选择喜欢的主题，进入对应的github获取主题源码
  3. 进入博客主目录，git clone 仓库地址 themes/主题名称（可自定义）
  4. 将_config.yml 下面的theme: landscape 修改为theme：主题名称
  5. 其他依赖包相关可参见主题所在github下的关于主题仓库的说明
  6. hexo clean –> hexo g –>hexo server

综上所述均为本地如何开发hexo，我们构建个人博客需要远程访问，接下来，讲述如何将github与hexo结合开发个人博客：

将本地博客发布在github上面

  2. 进入项目主目录打开_config.yml，在文中对应位置添加以下配置

>

  1. 打包hexo g

  2. 执行构建，在此之前需要安装构架依赖的组件：npm install hexo-deployer-git –save （如没有该依赖会报错：ERROR Deployer not found: git）

  3. 执行构建，hexo d或者hexo deploy 或者 hexo d -g

  4. 在github配置的页面访问路径下访问该博客
